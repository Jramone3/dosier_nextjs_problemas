# Informe Dosier Next.js Problemas

## Español
Este informe documenta los problemas más comunes encontrados en proyectos Next.js:  
- Configuración de rutas dinámicas.  
- Integración con APIs externas.  
- Manejo de errores en SSR y CSR.  

### Narrativa
El dosier busca ofrecer soluciones reproducibles y bilingües para facilitar el onboarding comunitario.

---

## English
This report documents the most common issues found in Next.js projects:  
- Dynamic route configuration.  
- Integration with external APIs.  
- Error handling in SSR and CSR.  

### Narrative
The dossier aims to provide reproducible bilingual solutions to facilitate community onboarding.
